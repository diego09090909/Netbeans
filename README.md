Tambien el programa nos permite gestionar usuarios mediane un login y un registro de nuevos usuarios.

Lo siguiente que se podra ver en este git es un segundo branch llamado "main" el cual tiene un backup del sistema para cuando teniamos problemas en los push a travez de Netbeans.

Por ultimo esta el Branch "jaxxcito-patch-1" se encuentra la base de datos actualizada llamada "bd_produccion_casino.sql", una carpeta llamada Dist la cual posee en su interior los ejecutables del proyecto funcional.

Para acceder a su funcionamiento previamente debera tener instalado XAMPP o MySQL en su computador para acceder a la base de datos anteriormente mencionada:

Pasos para ejecutar:
1- Descargue La base de datos en su computador
2- Descargue la carpeta dist
3- En su equipo ingrese a la carpeta Dist/lib, aqui debera dar doble click a los archivos ejecutables tipo .jar, partiendo por el mysql-conector y el AbsoluteLayout, por ultimo vuelva a la carpeta anterior de /Dist y de doble click en BdProduccionCasino para poder ejecutar el sistema.
